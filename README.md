# AWS Devops Production Project

## Overview
This project demonstrates an end-to-end DevOps workflow using AWS, Terraform, Jenkins, Docker, and Linux.
## Tech track
AWS(VPC,EC2,ALB,Cloudwatch)
Terraform(Infra as a code)
Jenkins(CI/CD)
Docker
Linux
GitHub
## Features
Infrastructure provisioning using Terraform
CI/CD pipeline with Jenkins
Dockerized application deployment
EC2 cost optimization using start/stop jobs
Monitoring via Cloudwatch
## Architecture
Custom VPC
EC2 app server
Application load balancer
Jenkins CI/CD
